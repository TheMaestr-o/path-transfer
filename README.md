Professional path transfer utility for Adobe Photoshop. Seamlessly transfers clipping paths between different file formats by matching filenames.

## Features

- Transfer paths from TIFF to JPG files
- Batch processing with folder selection
- Automatic filename matching
- Preserves path integrity
- Format-agnostic path transfer
- Detailed operation logging

## Installation

Copy `path-transfer.jsx` to your Photoshop Scripts folder:
- macOS: `~/Library/Application Support/Adobe/Adobe Photoshop [VERSION]/Presets/Scripts/`
- Windows: `C:\Program Files\Adobe\Adobe Photoshop [VERSION]\Presets\Scripts\`

## Usage

1. Open Photoshop
2. File → Scripts → Browse → Select `path-transfer.jsx`
3. Select source folder (TIFF files with paths)
4. Select destination folder (JPG files)
5. Script automatically transfers matching paths

## Requirements

- Adobe Photoshop CS6 or newer
- Source images with clipping paths
- Matching filenames between source and destination

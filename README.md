# PathTransfer

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Photoshop](https://img.shields.io/badge/Photoshop-CS6+-ff00ff)

Transfer clipping paths from TIF files to JPG files by filename matching.

## Features

- Automatic file matching by name
- Batch path transfer
- Format preservation
- Safe operation (TIF not modified)

## Installation

Copy to Photoshop Scripts folder

## Usage

File → Scripts → Browse → Select source and destination folders
